# CustomPartyGlow

World Map addon that adds pulsing, class-coloured glow blips for party and raid members.

[Originally made for WoW Retail](https://www.curseforge.com/wow/addons/custompartyglow), ported to vanilla 1.12.

## Features

- Class-coloured glow overlay on each party/raid member's map position
- Pulsing animation for visibility
- Adjustable icon size via in-game slider
- Saved settings persist between sessions
- Compatible with [Magnify](https://github.com/lookino/Magnify) (handles zoom scale and strata changes)

## Installation

Copy the contents to

```
Interface/AddOns/CustomPartyGlow/
```

Make sure the folder contains `CustomPartyGlow.toc` and `CustomPartyGlow.lua` at the top level.

## Usage

- `/cpg` or `/custompartyglow` to open the size slider
- A small button also appears on the top-right corner of the world map
- Blips only appear when you are in a party or raid
- Setting size to 16 hides all blips (acts as a toggle)
